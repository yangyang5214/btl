# btl

> beer's tools

```shell
go install github.com/yangyang5214/btl@latest 
```

## gist

Download the github gist to a local file, including images

```shell
btl gist <gist_id>
btl gist 7f903d466483c1dcfa62e131bcbb7917
```

## json2csv

json lines covert to CSV format

```shell
btl json2csv  -f xxx.json
```
