# btl

> beer's tools

```shell
go install github.com/yangyang5214/btl@latest 
```

### cmds

```shell
~ btl
 
Usage:
  btl [command]

Available Commands:
  amap_replaying   amap replay
  completion       Generate the autocompletion script for the specified shell
  csv2mind         csv file covert to mind map
  date             date format
  email            send email
  fit2gpx          fit to gpx file
  garmin_upload    garmin_upload
  gist             download a gist to local md file
  gpx2Json         gpx file to json
  gpx2kml          gpx to kml file
  gpx2video        gen video by gpx
  gpx_amap         gen image by amap
  gpx_export       export gpx from apps
  gpx_merge        merge gpx files
  gpx_rm_extension gpx remove extension
  gpx_speed        customize gpx speed
  gpx_zip          zip gpx file to limited size.(remove some points)
  heatmap          heatmap for merge_gpx
  help             Help about any command
  json2csv         json to csv
  json2mind        json lines file transform to md file, then convert to mind map
  kill_chrome      kill all chrome process(chromium)
  markerPoint      amap marker point
  screenshot       screenshot for input html
  xhs              xhs images

Flags:
  -f, --file string   input file
  -h, --help          help for btl
  -p, --pwd string    password
  -t, --toggle        Help message for toggle
  -u, --user string   username. email

Use "btl [command] --help" for more information about a command.
```
